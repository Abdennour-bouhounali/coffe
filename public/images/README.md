# Médias

Déposer ici les images finales optimisées en AVIF ou WebP.

Conventions recommandées :

- `home/hero-maison-saha.avif`
- `menu/saha-benedict.webp`
- `menu/pistachio-cloud.webp`
- `gallery/interieur-01.webp`

Chaque image doit disposer d’une largeur et d’une hauteur adaptées à son usage,
ainsi que d’une alternative textuelle définie dans les données éditoriales.
